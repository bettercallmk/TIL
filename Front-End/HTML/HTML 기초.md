# HTML 기초

## 1. HTML 개요

**HTML**(Hyper Text Markup Language)는 웹 페이지를 구성하는 **기본 구조를 정의하는 마크업 언어**이다. 텍스트, 이미지, 링크, 폼, 버튼, 제목, 단락 등을 **HTML 태그(tag)**라는 마크업 요소를 사용하여 콘텐츠의 의미와 구조를 표시한다.



## 2. HTML 기본 태그

### 2-1 단락 태그 `<p>`

텍스트를 단락(paragraph)을 정의하는 태그이다. 텍스트를 문단 단위로 구분할 때 사용된다.

```html
<p>문장 1 입니다.</p>
<p>문장 2 입니다.</p>
```

출력 결과 :

문장 1 입니다.
문장 2 입니다.
